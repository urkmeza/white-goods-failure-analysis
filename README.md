# white-goods-failure-analysis

Project Title: Predictive Quality Analysis in Appliance Manufacturing

Business Problem: High warranty costs due to "Pump Failures" in Washing Machine Model-X between months 24-30.

Objective: Identify if failures are linked to specific production batches (Shift/Line) or environmental factors (Water Hardness/Usage Frequency).

Tech Stack: > * SQL: Data joining and filtering.

Python (Pandas/Scipy): Data cleaning and 3-Sigma outlier detection.

R (Tidyverse/Ggplot2): Hypothesis testing (ANOVA) and statistical visualizations.

Tableau: Executive Dashboard for Root Cause Analysis.
